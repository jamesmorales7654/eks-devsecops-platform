# 🚀 EKS DevSecOps Platform

![Terraform CI](https://github.com/jamesmorales7654/eks-devsecops-platform/actions/workflows/terraform-eks.yml/badge.svg)

This project provisions a secure, production-grade Amazon EKS cluster using Terraform and enforces DevSecOps best practices through:
- 🧱 **Infrastructure-as-Code validation**
- 📏 **TFLint static analysis**
- 🛡️ **Checkov security policy scanning**
- ✅ **Automated GitHub Actions CI on every pull request**

> 🧠 All changes are tested in an AWS sandbox environment and must pass automated security gates before merging to `main`.
